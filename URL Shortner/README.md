# URL Shortner

This is a beautiful python script used to create the url shortner with different url api's.  
The Following URL Shortners you can short your link are :-

1. TinyUrl
2. Clck.ru
3. Da.gd
4. Is.gd
5. Os.db
6. Ow.ly
7. Create Your Own URL
